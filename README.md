# 校园活动报名与签到管理系统

## 1. 项目概述
本项目用于《软件项目管理》课程综合实验，目标是在统一题目背景下完成项目初始化、WBS 分解、简化进度计划、配置管理与协作提交。

## 2. 团队成员与角色
| 姓名 | 角色 | 主要职责 |
|---|---|---|
| 成员A | 项目经理 | 统筹进度与最终提交 |
| 成员B | 计划负责人 | 负责 WBS 与进度计划文档 |
| 成员C | 配置管理员 | 负责仓库、分支和合并管理 |
| 成员D | 协作成员 | 参与文档完善、分支提交与冲突解决 |

## 4. 分支约定
- `main`：最终提交分支
- `dev`：阶段集成分支
- `feature-*`：成员个人功能分支
